FMOD_DSP_SEND

Parameter types for the FMOD_DSP_TYPE_SEND DSP.

C/C++ Syntax

typedef enum {
  FMOD_DSP_SEND_RETURNID,
  FMOD_DSP_SEND_LEVEL
} FMOD_DSP_SEND;

Values

FMOD_DSP_SEND_RETURNID

(Type:int) - ID of the Return DSP this send is connected to (integer values only). -1 indicates no connected Return DSP. Default = -1.

FMOD_DSP_SEND_LEVEL

(Type:float) - Send level. 0.0 to 1.0. Default = 1.0
