Firelight Technologies FMOD Studio API

FMOD_COMPLEX

Complex number structure used for holding FFT frequency domain-data for FMOD_FFTREAL and FMOD_IFFTREAL DSP functions.

C/C++ Syntax

typedef struct {
  float real;
  float imag;
} FMOD_COMPLEX;

Members

real

Real component

imag

Imaginary component
